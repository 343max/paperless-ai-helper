# paperless-ai-helper

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/main.ts
```

This project was created using `bun init` in bun v1.0.27. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

```sh
#!/usr/bin/env bash

curl -X "POST http://ai-helper:8080/process/${DOCUMENT_ID}"
```
